# My Blog

> Hugo version: [v0.97.3](https://github.com/gohugoio/hugo/releases/tag/v0.97.3) (See env variable HUGO_VERSION on vercel [Link](https://discourse.gohugo.io/t/vercel-tips/34766))

# Some commands

## Create a new post

```bash
hugo new posts/my-new-post.md
```

## Run the server

```bash
hugo server -D
```

## Build the site

```bash
hugo
```
